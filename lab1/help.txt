Для загрузки PDF нужно нажать кнопку Загрузить PDF.
Для поиска словоформы/лексемы в таблице, введите эту словоформы/лексему в поиск.
Для того, чтобы экспортировать файл в формате txt/csv перейдите в меню файл и выберите соответствующий вариант сохранения.
Для выхода из приложения нажмите крестик.
Для выбора варианта сортировки перейдите в меню сортировка и выберите удовлетворяющий Вас вариант.
